# PhotoEditor


## Features
- [x] Cropping 
- [x] Rotate
- [x] Flip Horizontal
- [x] Flip Verical


Download the app in zip or run 

```
git clone https://github.com/joysonjson/PhotoEditor.git
```

Then, run the following command:

```bash
$ pod install
```

Open PhotoEditor.xcworkspace , Build and run the app.
